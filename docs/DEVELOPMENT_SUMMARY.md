# Hao-Code 开发总结报告

## 1. 项目概述

Hao-Code 是一个基于 Wails v3 + Vue 3 技术栈构建的现代化跨平台代码编辑器。本项目旨在复刻 VSCode 的核心功能，同时提供 Idea 级别的 Git 可视化体验和 LSP 智能辅助。

## 2. 技术架构

- **前端**: Vue 3 (Composition API) + TypeScript + Naive UI + Monaco Editor
- **后端**: Go (Wails v3 Runtime) + JSON-RPC 2.0
- **通信**: Wails Events & Bindings (自动类型同步)
- **状态管理**: Pinia
- **核心依赖**:
  - `monaco-editor`: 编辑器内核
  - `xterm.js`: 终端渲染
  - `go-git/libgit2`: Git 逻辑处理
  - `gopls`: Go 语言服务器
